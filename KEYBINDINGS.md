# Keybindings

| Prefix | Key | Action | Mode |
|--------|-----|--------|------|
| **File explorer** |||
| `<leader>pv` | File browser (Telescope) | n |
| **LSP** |||
| `gd` | Go to definition | n |
| `K` | Hover documentation | n |
| `<leader>vws` | Workspace symbols | n |
| `<leader>vd` | Line diagnostics | n |
| `[d` | Previous diagnostic | n |
| `]d` | Next diagnostic | n |
| `<leader>vca` | Code action | n |
| `<leader>vrr` | References | n |
| `<leader>vrn` | Rename | n |
| `<C-h>` | Signature help | i |
| **Telescope** |||
| `<leader>pf` | Find files | n |
| `<C-p>` | Git files | n |
| `<leader>ps` | Grep string (prompt) | n |
| `<leader>pg` | Live grep | n |
| `<leader>ph` | Help tags | n |
| `<leader>pk` | Keymaps | n |
| `<leader>po` | Recent files | n |
| `<leader>pd` | Diagnostics | n |
| `<leader>pb` | File browser | n |
| `<leader>pp` | Projects | n |
| **Format** |||
| `<leader>f` | Format buffer (conform) | n |
| **Trouble** |||
| `<leader>xx` | Toggle diagnostics | n |
| `<leader>xX` | Buffer diagnostics | n |
| `<leader>xl` | Location list | n |
| `<leader>xq` | Quickfix list | n |
| **Git** |||
| `<leader>gs` | Git status (fugitive) | n |
| **Terminal** |||
| `<F7>` | Toggle floating terminal | n |
| **Undo** |||
| `<leader>u` | Undo tree | n |
| **Buffers** |||
| `<Tab>` | Next buffer | n |
| `<S-Tab>` | Previous buffer | n |
| **Todo** |||
| `]t` | Next todo comment | n |
| `[t` | Previous todo comment | n |
| **Debug (DAP)** |||
| `<leader>db` | Toggle breakpoint | n |
| `<leader>dc` | Continue | n |
| `<leader>do` | Step over | n |
| `<leader>di` | Step into | n |
| `<leader>dO` | Step out | n |
| `<leader>dd` | Toggle DAP UI | n |
| `<leader>dt` | Terminate | n |
| **Test (Neotest)** |||
| `<leader>tn` | Run nearest test | n |
| `<leader>tf` | Run file tests | n |
| `<leader>ts` | Toggle summary | n |
| `<leader>to` | Toggle output panel | n |
| `<leader>ta` | Run all tests | n |
